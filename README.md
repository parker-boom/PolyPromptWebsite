# Poly Prompt

Landing page for Poly Prompt — an OpenAI hackathon at Cal Poly (Feb 13–15, 2026).

## Run Locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Build & Deploy

```bash
npm run build
```

Outputs to `.next/` — deploy to Vercel, Netlify, or any static host.
